#__LUMENGINE__
## **HOW TO BUILD**

1. Download zip file
2. Unpack zip file
3. Run "init.bat"
4. "build" folder will show up
5. Go into "build" folder and run LumEngine.sln
6. Done

## **NOTE: If projects doesn't compile, set LumEngine project as startup project (it will be fixed soon)**
